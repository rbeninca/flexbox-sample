Properties for the Children (flex items)

flex item order

A propriedade order no Flexbox é usada para controlar a ordem em que os itens flexíveis aparecem dentro do contêiner flexível. Por padrão, todos os itens flexíveis têm um valor de order de 0, o que significa que eles aparecerão na ordem em que são colocados no código HTML. Ao ajustar o valor da propriedade order, você pode reorganizar os itens sem alterar o HTML. Os itens são ordenados de acordo com o valor numérico da propriedade order, do menor para o maior. Itens com o mesmo valor de order aparecerão na ordem em que aparecem no código HTML.

flex item shrink

Propriedade define como um item flexível deve encolher se o contêiner flexível não tiver espaço suficiente para acomodar todos os itens em seu tamanho natural.
O valor padrão de flex-shrink é 1, o que significa que o item está autorizado a encolher se necessário. Se você definir flex-shrink como 0, o item não encolherá além do seu tamanho base, mesmo se o contêiner flexível não tiver espaço suficiente para acomodá-lo.

Pontos Importantes

flex-shrink aceita qualquer número não negativo como valor. O encolhimento é proporcional ao valor de flex-shrink. Por exemplo, um item com flex-shrink: 2
encolherá duas vezes mais rápido do que um item com flex-shrink: 1 quando o espaço é insuficiente. A propriedade flex-shrink só tem efeito quando o espaço é insuficiente; ela não faz nada se houver espaço extra no contêiner.

flex item basis

A propriedade flex-basis no Flexbox define o tamanho inicial de um item flexível antes de qualquer espaço ser distribuído de acordo com as propriedades flex-grow e flex-shrink. Em outras palavras, flex-basis é usado para definir o tamanho base de um item flexível.
